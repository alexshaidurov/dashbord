# Prep Dashboard PRO — Установка на GitHub Pages

## Шаг 1 — GitHub репозиторий

1. Зайди на [github.com](https://github.com) → Sign up (если нет аккаунта)
2. Нажми **New repository** (зелёная кнопка)
3. Название: `prep-dashboard` (или любое)
4. Выбери **Public**
5. Нажми **Create repository**

## Шаг 2 — Загрузка файлов

В новом репозитории нажми **uploading an existing file** и загрузи ВСЕ 5 файлов:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

Нажми **Commit changes**.

## Шаг 3 — Включить GitHub Pages

1. В репозитории → **Settings** → слева **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)**
4. Нажми **Save**
5. Через ~2 минуты появится ссылка: `https://ВАШ_НИК.github.io/prep-dashboard`

## Шаг 4 — Supabase авторизация

1. Открой ссылку на дашборд
2. ⚙️ Настройки → «Облачная синхронизация»
3. Введи email → «Отправить Magic Link»
4. Открой письмо → скопируй 6-значный код → вставь → «Подтвердить»
5. Нажми **⬆ Выгрузить в Supabase** — первая синхронизация

## Шаг 5 — Установка на домашний экран iPhone

1. Открой ссылку в **Safari** (не Chrome!)
2. Нажми кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Прокрути вниз → **На экран «Домой»**
4. Название: PrepPRO → **Добавить**

Теперь дашборд работает как нативное приложение с иконкой на рабочем столе.

## Автосинхронизация

После входа в аккаунт включи **«Автосинхронизация каждые 10 минут»** — данные будут автоматически выгружаться в Supabase. На другом устройстве зайди под тем же email и нажми **⬇ Загрузить из Supabase**.
